# 🚀 Космические исследователи

Космический арканоид на Pygame, где нужно уничтожать космический мусор, уклоняясь от него.

---

## 🎮 Управление

| Клавиша | Действие |
|---------|----------|
| ← → | Движение корабля влево/вправо |
| Пробел | Стрельба лазером |
| ESC | Пауза / Выход из игры (на экране Game Over) |
| R | Перезапуск игры после окончания |

---

## 🕹️ Игровая механика

- **Цель игры** — уничтожать космический мусор, не допуская его столкновения с кораблём или падения за нижнюю границу экрана.

- **Счёт:** за каждый уничтоженный объект мусора начисляется **10 очков**.

- **Жизни:** у игрока **3 жизни**. Жизнь теряется в двух случаях:
  - при столкновении корабля с мусором
  - если мусор улетел за нижнюю границу экрана

- **Потеря жизни:** после потери жизни текущая армия мусора полностью удаляется, и на её месте появляется **новая армия** со случайным количеством объектов (от 20 до 50). Счёт при этом продолжает накапливаться.

- **Уничтожение всей армии:** если игрок уничтожил весь мусор лазерами, сразу же появляется новая армия со случайным количеством объектов (от 5 до 100).

## ⚙️ Установка и запуск

### 1. Установите Python

Убедитесь, что у вас установлен **Python версии 3.8 или выше**.

### 2. Установите Pygame

Откройте терминал (командную строку) и выполните команду:

```bash
pip install pygame
3. Подготовьте изображения
Создайте в папке с проектом папку image

Положите в неё изображение корабля с именем pyshka.png (размер 50×50 пикселей)

Создайте внутри image папку musor

Положите туда все PNG‑картинки мусора (каждая размером 50×50 пикселей). Имена файлов могут быть любыми.

4. Запустите игру
bash
python space_in.py
